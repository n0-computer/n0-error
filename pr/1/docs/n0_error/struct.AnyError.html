Struct AnyError

Source 
pub struct AnyError(/* private fields */);
Expand description

Type-erased error that can wrap a StackError or any std::error::Error.

StackErrors have a blanket impl to convert into AnyError, while preserving their call-site location info.

Errors that implement std::error::Error but not StackError can’t convert to AnyError automatically. Use either AnyError::from_std or crate::StdResultExt::anyerr to convert std errors into AnyError.

This is necessary unfortunately because if we had a blanket conversion from std errors to AnyError, this blanket conversion would also apply to StackErrors, thus losing their call-site location info in the conversion.
